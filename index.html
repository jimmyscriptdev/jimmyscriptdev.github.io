Server Uptime Populator

Populate Airtable with server information using the uptime command for real-time insights.


#!/bin/bash
# Script: airtable_uptime.sh
curl -s -X POST https://api.airtable.com/v0/$__DATABASE/$__TABLE \
-H "Authorization: Bearer $__APIKEY" \
-H "Content-Type: application/json" \
--data '{ "records": [ { "fields": { "Name": "$__HOST", "Notes": "$(uptime)" } } ] }'

System Monitor

Monitor memory and CPU usage in real time with this efficient loop-based Bash script.


#!/bin/bash
# Script: system_monitor.sh
while true; do
    echo "Memory Usage: $(free -m | awk 'NR==2{printf "%.2f", $3*100/$2 }')%"
    echo "CPU Load: $(top -bn1 | grep "Cpu(s)" | sed "s/.*, *\([0-9.]*\)%* id.*/\1/" | awk '{print 100 - $1}')%"
    sleep 5
done
